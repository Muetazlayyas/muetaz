Reconnaissance: Open-source intelligence (OSINT) tools, online search engines, and social media monitoring tools.
Weaponization: Integrated development environments (IDEs), code review tools, and static code analysis tools.
Delivery: Malware distribution platforms, exploit kits, or compromised update channels.
Exploitation: Exploit frameworks, vulnerability scanners, and penetration testing tools.
Installation: Remote access trojans (RATs), command-line utilities, or custom malware.
Command and Control: Traffic analysis tools, network security monitoring (NSM) tools, and intrusion detection/prevention systems (IDS/IPS).
Actions on Objectives: Data exfiltration tools, lateral movement frameworks, and privilege escalation tools.
